# ThreadQL Documentation

Welcome to the comprehensive documentation for ThreadQL - a natural language to SQL query system.

## What is ThreadQL?

ThreadQL is a Laravel-based application that allows users to interact with databases using natural language queries. It translates natural language requests into SQL queries and executes them against tenant databases, providing results in a Slack-friendly format.

## Key Features

- **Natural Language Processing**: Convert natural language to SQL queries
- **Tenant Support**: Multi-tenant database architecture
- **Slack Integration**: Native Slack app integration
- **MCP Tools**: Model Context Protocol tools for AI integration
- **CSV Export**: Export query results to CSV files
- **Security**: Internal network restriction and query validation

## Quick Start

To get started with ThreadQL:

1. **Installation**: Follow the installation guide
2. **Configuration**: Set up your database and Slack integration
3. **Usage**: Start using the MCP tools or HTTP API
4. **Integration**: Connect to your applications via MCP
